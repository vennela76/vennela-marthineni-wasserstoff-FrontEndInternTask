## set up and run
```bash
cd ./app
```
write to all 4 services. 
```bash
cd ./frontend (eg. of 1 service)
npm i 
```
now open docker desktop.
after write in terminal

```bash
cd ../..
docker-compose up -d
```

then run 
```bash 
npm run dev
```

which start 4 different services.
